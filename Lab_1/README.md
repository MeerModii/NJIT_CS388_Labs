# Tap Counter App

A simple Android application built with Kotlin and XML-based layouts for Unit 1 Lab.

## Demo
https://github.com/user-attachments/assets/bc04cd93-245d-4cd9-91c3-31e188e2c86e

## Features
- Click the "TAP" button to increase the counter.
- The count updates in real-time on the screen.
- At 100 taps, an "Upgrade" button appears that changes the increment value from +1 to +2.
- Custom dark background
- Includes a penguin image at the bottom of the layout.
